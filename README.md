# FAST GPA & CGPA Calculator

A sleek, web-based GPA and CGPA calculator designed specifically for students, featuring a modern neon-blue user interface. This tool helps students track their academic performance by calculating Semester GPA (SGPA) and Cumulative GPA (CGPA) based on university grading scales.

## üöÄ Features

* **SGPA Calculation:** Input course names, credit hours, and grades to get an instant semester breakdown.
* **CGPA Tracking:** Manage multiple semesters to calculate your overall cumulative standing.
* **Interactive UI:** Fully responsive design with a "Dark Mode" aesthetic and glowing accents.
* **Reference Tables:** Quick access to Grade and Point distribution tables for accuracy.
* **Dynamic Row Management:** Add or delete courses and semesters on the fly.
